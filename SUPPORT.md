# Support

Thanks for checking out Loop Engineering.

## Questions

Use GitHub Issues for:

- Documentation questions.
- Audit workflow questions.
- Reproducible bugs.
- Feature ideas.
- Launch-readiness improvements.

## Security

Do not open public issues for sensitive vulnerabilities.

Read [SECURITY.md](SECURITY.md) for the disclosure process.

## Scope

This repo currently provides a production-quality foundation, audit workflow, and engineering standards. Product-specific support will be documented when application code is added.
